## Project Overview

This project focuses on developing a robust option pricing framework for European-style options by combining analytical and computational approaches. The primary objective is to implement and compare two widely used pricing methodologies: the Black–Scholes model and Monte Carlo simulation. The Black–Scholes model will be developed from first principles, beginning with the stochastic modeling of asset prices and culminating in the derivation and solution of the Black–Scholes partial differential equation (PDE). Its closed-form analytical solution will be implemented to compute the theoretical fair value of European call and put options. To complement the analytical approach, a Monte Carlo simulation framework will be developed to estimate option prices through stochastic sampling of asset price paths. By comparing the analytical Black–Scholes solution with Monte Carlo estimates, the project will evaluate the accuracy, efficiency, and practical applications of each method. 

The project is organized into two phases:

**Phase I** develops the mathematical foundations—including stochastic processes, Itô calculus, partial differential equations, and numerical analysis.
**Phase II** translates these mathematical models into Python implementations for simulation, pricing, visualization, and validation.
